## 代码逻辑
step1：先定义一个结构体（对象），结构体定义一些基本属性字段+基本属性转换后的保存字段
step2：给结构体（对象）定义一个构造函数来初始化结构体（对象）
step3：定义一个Transformer的interface，接口中定义Transform方法
step4：在每个具体处理业务里实现Transformer接口的Transform方法：给结构体（对象）定义一个Transform方法

## 运行
```
./transformation
```
